# Realm of Shadows — ChatGPT Image Prompts
# All 55 background images

## HOW TO USE THIS

1. Open ChatGPT (must have image generation / DALL-E access)
2. Upload Paladin.png and Spellblade.png from your sprites folder as reference images
3. For each prompt below, paste the STYLE PREFIX first (it's the same every time), then paste the specific prompt
4. Download each image, rename it exactly as shown, place it in the correct folder
5. After each batch, run the game to verify it looks right

---

## STYLE PREFIX — paste this before every single prompt:

> Pixel art in the exact same visual style as the attached reference images. Chunky pixel blocks, hard stepped edges, flat colour fills with 3-4 tones per material, dark outlines separating major forms. Atmospheric dark fantasy. No anti-aliasing, no soft gradients, no smooth curves — pure pixel art. The scene should fill the entire canvas with no white space or borders.
