# SESSION 1: Town Exterior Backgrounds (11 images)

## HOW TO DO THIS JOB

You will generate 11 images using ChatGPT's image generation. Follow these steps for EVERY image:

**STEP 1:** Open ChatGPT in the browser at https://chatgpt.com
**STEP 2:** Start a new conversation
**STEP 3:** Paste the image prompt (provided below for each image)
**STEP 4:** Wait for ChatGPT to generate the image
**STEP 5:** Download the image by clicking the download button on the generated image
**STEP 6:** Move and rename the downloaded file to the exact path shown
**STEP 7:** Confirm the file exists at the correct path before moving to the next image

Do NOT use Python to draw images. Do NOT use PIL or any drawing library.
Use ChatGPT image generation ONLY.
Do each image one at a time. Do not move to the next until the current one is saved.
